## ğŸ“¦ Cara Menggunakan

1. **Clone atau Download Repository**

   ```bash
   git clone https://github.com/PMII-Rayon-Inklusif-Justicia-UNESA/website.git
   ```

2. **Buka di Browser**

   - Buka file `index.html` di browser untuk melihat halaman beranda
   - Atau gunakan Live Server di VS Code untuk pengembangan

3. **Kustomisasi Konten**

   - Edit file HTML di folder `page/` untuk mengubah konten
   - Ganti gambar di folder `assets/img/`
   - Sesuaikan warna dan style di `assets/css/`

4. **Deploy Website**

   - **GitHub Pages**: Push ke repository GitHub dan aktifkan GitHub Pages
   - **Netlify/Vercel**: Upload folder project atau connect via Git
   - **Hosting lain**: Upload semua file ke server hosting
